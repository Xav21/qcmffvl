A propos : QCM FFVL version 1.3 (dev)
Auteur : Jérémy Ruffet (sat@airnux.fr)

Cet application web est open source et accessible sur github (https://github.com/sativouf/qcmffvl)

Elle a été conçue pour être multi-support (mobile / tablette / PC / MAC...).

La totalité du questionnaire n'est chargé qu'une seule fois en mode révision.
Cela veut dire qu'il vous suffit de charger la page sur votre mobile avant de partir, et vous pourrez vous faire d'innombrables QCMs FFVL, sans avoir jamais besoin de réseau !

Cette "appli web" a été testée comme fonctionnant correctement avec Chrome et Firefox.
Merci d'utiliser un de ces 2 navigateurs si vous expérimentez des soucis.

Pour l'instant, seul le navigateur Chrome est compatible avec le mode "Examen Papier", cela sera probablement amené à évoluer dans le futur.

N'hésitez pas à m'envoyer des rapports de bugs ou des suggestions par mail.
Et bien entendu, les bières de remerciement sont les bienvenues ! :)
